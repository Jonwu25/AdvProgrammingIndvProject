# Advanced Programming Individual Project: Evolution Simulation
This project is a evolution simulation, with a tile-based map. Creatures, represented as circles, will be given information on the environment and other creatures, including where the food is. However, the creature will have an essentially random AI, and will eventually learn to find food. The user will be able to adjust the environment, specific AI parameters like the mutation rate, and visual parameters like how fast the program is running.

# Run Instructions from the executable:

1. Download the repository as a zip file
2. Unzip and navigate to your operating system in the src folder
3. Run the executable

# Run Instructions from the Terminal:

1. Download the repository as a zip file.
2. Navigate to the `AdvProgrammingIndvProject-main` folder in console
3. Type `javac -cp lib/core.jar src/*.java` to compile
    * For Windows, type `java -cp "lib/core.jar;src" Display` to run
    * For Mac and Linux, type `java -cp "lib/core.jar:src" Display` to run
